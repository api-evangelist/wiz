# Wiz GraphQL API

GraphQL API for querying and managing cloud security inventory, issues, vulnerabilities, misconfigurations, identities, controls, and reports across the Wiz security graph. Authentication uses OAuth 2.0 client credentials with tenant-specific endpoints.

**Endpoint:** https://api.us1.app.wiz.io/graphql

**Documentation:** https://docs.wiz.io

**GraphQL Properties:**

- GraphQL: https://api.us1.app.wiz.io/graphql
